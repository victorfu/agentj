Connect to LINE

Follow this step-by-step guide to synchronize your application with the LINE Developers Console. This allows your bot to receive and respond to messages in real-time.

content_copy
Step 1

Copy Your Webhook URL

This URL is unique to your application. You'll need to paste this into the LINE Developers Console to enable message forwarding.

https://api.lineservice.com/webhook/v1/9823-ajkd-8232-0921
login
Step 2

Access LINE Developers Console

Log in to your account at developers.line.biz. Select the provider and the Messaging API channel you want to connect.

domain
Example of LINE Console Dashboard interface
settings_input_component
Step 3

Paste URL and Enable Webhook

Go to the Messaging API tab. Find the "Webhook settings" section, paste the URL copied in Step 1, and make sure the "Use webhook" toggle is switched ON.

Step 4

Verify Connection

Click the "Verify" button in the LINE Console. If the setup is correct, you'll see a success message. Your integration is now active!

info Pro Tips

  • • Ensure your channel has "Messaging API" permissions enabled.
  • • Double check that no white spaces were copied with the URL.
  • • Wait up to 30 seconds for LINE servers to propagate changes.
